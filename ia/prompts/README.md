# Prompts de workflow

* `create-task.md`: convertir necesidad a tarea.
* `implement-next-task.md`: ejecutar tarea aprobada.
* `review-current-diff.md`: revisar cambios.
* `close-session.md`: persistir estado.

Mientras no exista MCP, cada prompt enumera lectura mínima. Al instalar `iaWorkflow`, usar `ia_validate` e `ia_get_context` primero.

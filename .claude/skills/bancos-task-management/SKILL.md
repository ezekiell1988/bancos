# Gestión de tareas Bancos

Usar al crear, aprobar, trabajar, bloquear o cerrar tareas. Leer `ia/04_tasks.md`, `ia/04_tasks/current.md` y `ia/templates/task-template.md`.

Crear IDs `TASK-{INICIALES}-{ÁREA}-{NN}`. Registrar primero como `Borrador`; mover a `Lista` solo con alcance, aceptación, riesgo, validación y aprobación completos. Riesgo alto exige aprobación explícita. Al completar, actualizar plan/progreso y archivar resumen mensual.
